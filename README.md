## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

Basic knowledge of React JS and CSS.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/VaniaTavares/gericht-restaurant.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
